`pd-orchidea` allows to use Orchidea-SOL samples through MIDI.


> [!WARNING]  
> `pd-orchidea` is distributed as part of the [`xlab`](https://github.com/charlesneimog/pd-xlab) library. 


## Download and Install

1. Create a new Pure Data patch.
2. Go to Help → Find Externals → and Search for `pd-xlab`.
3. Create a new object `declare -lib xlab`.
4. Create one of the objects.
